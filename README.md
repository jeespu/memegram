# Memegram Web App - School Project [](https://joosia.github.io/meemi/)
A social web application for sharing, rating and commenting memes, thus the name Memegram. 
## Features:
* Meme feed
* Register an account
* Post a meme
* Comment and rate
## Optional:
* Profile Settings Page (reset password, change username) 
* Admin account
* Memegenerator
